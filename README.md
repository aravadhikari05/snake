# Snake

A simple snake game that I made for fun

![screenshot](screenshots/game.png)

#### Features

* Score
* Audio (plinggg)
* Grid
* Colors

#### How to play

Pretty self-explanatory but: 
* WASD to move
* Eating apples makes snake longer and increases score
* Hitting walls or the snake tail restarts the game
